FEATURE 1

Auto Clean Mode

Instantly strip away cookie banners, newsletter popups, and sticky ads that ruin automated screenshots.

Under the Hood

We use a combination of EasyList-inspired filtering and heuristic detection to identify high-z-index overlays and remove them before the shutter snaps.

"clean": true
FEATURE 2

Smart Stabilization

Never capture a half-loaded page again. We wait for fonts, network requests, and layout shifts to settle.

Under the Hood

Our engine monitors the DOM for changes. If a lazy-loaded image or a slow font causes a layout shift, we reset the timer and wait for true stability.

"wait": "smart"
FEATURE 3

Animation Freezing

Stop CSS animations and transitions in their tracks to get consistent, non-blurry results.

Under the Hood

We inject a specialized CSS runtime that forces all transition and animation durations to 0s, effectively freezing the page at the perfect frame.

"freezeAnimations": true
FEATURE 4

Social Card Templates

Generate beautiful social cards, certificates, and invoices from HTML templates with dynamic data.

Under the Hood

Pass your data as JSON and we'll inject it into our high-performance HTML templates. Perfect for dynamic OG images or PDF generation.

"template": "social-card"
